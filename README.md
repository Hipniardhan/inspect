# Inspection Report — Static GitHub Pages

Versi statis dari aplikasi Form Inspeksi Kendaraan.

## Menjalankan secara lokal

Buka `index.html` langsung di browser, atau gunakan ekstensi Live Server di VS Code.

## Publikasi ke GitHub Pages

1. Upload seluruh isi folder ini ke repository GitHub.
2. Buka **Settings → Pages**.
3. Pada **Build and deployment**, pilih **Deploy from a branch**.
4. Pilih branch **main** dan folder **/(root)**.
5. Klik **Save**.

Link biasanya tersedia dalam beberapa menit dengan format:

`https://USERNAME.github.io/NAMA-REPOSITORY/`

## Catatan

- Tidak membutuhkan PHP, Laravel, database, Composer, atau Node.js.
- Draft disimpan pada Local Storage browser.
- Tombol Print menggunakan dialog print bawaan browser.
- Mode Angka mendukung nilai desimal berkoma, misalnya `0,17`.
